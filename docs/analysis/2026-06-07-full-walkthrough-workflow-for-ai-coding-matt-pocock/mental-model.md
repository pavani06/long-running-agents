## 1. Project Goals

- Build and manage long-running AI agent workflows through `.opencode/`, Node >= 20.18.0, ESLint, runtime state in `.runtime/` and `artifacts/`, and accepted decisions under `docs/decisions/` (`AGENTS.md:7`, `AGENTS.md:9`, `AGENTS.md:10`, `AGENTS.md:11`, `AGENTS.md:12`).
- Serve as a knowledge base and curriculum for AI systems that operate reliably for hours, days, or as long as the task requires without losing context, planning ability, or quality judgment (`README.md:3`).
- Teach the structural failure model for long-running agents: context loss, fragile planning, and blind self-evaluation, then teach harnesses as the support structures that manage context, decompose work, and separate generation from evaluation (`README.md:7`, `README.md:9`, `README.md:10`, `README.md:11`, `README.md:13`).
- Target business builders with agentic-system skill, from beginners through production operators who want more reliable systems (`README.md:15`, `README.md:17`).
- Deliver a 12-week curriculum with 4 levels, 8 core concepts, and 35+ diagrams applied to KODA, the WhatsApp supplement-sales agent used as the practical case (`curriculum/README.md:11`, `curriculum/README.md:13`, `curriculum/README.md:19`, `curriculum/README.md:33`, `curriculum/README.md:34`).
- Maintain documentation governance through a strict precedence chain: accepted ADRs, active canonical docs, validated evidence, analyses, archive docs, then READMEs and operational summaries (`docs/system-of-record.md:14`, `docs/system-of-record.md:16`, `docs/system-of-record.md:17`, `docs/system-of-record.md:18`, `docs/system-of-record.md:19`, `docs/system-of-record.md:20`, `docs/system-of-record.md:21`; `AGENTS.md:64`, `AGENTS.md:66`, `AGENTS.md:68`, `AGENTS.md:69`, `AGENTS.md:70`, `AGENTS.md:71`, `AGENTS.md:72`, `AGENTS.md:73`).
- Operationalize agent work with issue claim, isolated worktree, execution brief, validation, draft PR, second-agent review, explicit approval before merge, and cleanup (`.opencode/skills/issue-start/SKILL.md:12`, `.opencode/skills/issue-start/SKILL.md:14`, `.opencode/skills/issue-review/SKILL.md:12`, `.opencode/skills/issue-review/SKILL.md:14`, `.opencode/skills/issue-finish/SKILL.md:12`, `.opencode/skills/issue-finish/SKILL.md:14`).
